Hallo, ik ben Bram ik ben 20 jaar oud en een student bij hogeschool PXL en studeer graduaat programmeren. Ik ben op het moment al 2 jaar aan het verder studeren. In dit portfolio ga ik noteren wat ik dit jaar heb gezien bij het vak WPL en ga ik ook reflecteren voor elke opdracht of toets die ik bij WPL heb gemaakt. Hierdoor kan ik zelf zien wat ik slechter en beter deed en dus voor wat ik nog moet oefenen.

Waarom heb ik voor deze richting gekozen?

Toen ik op het einde van mijn 6de jaar van het middelbaar zat was ik nog niet uit wat ik wow gaan verder studeren want ik wow wel verder studeren, maar ik wist niet wat. Maar toen kwamen opeens een heel deel oud-leerlingen terug naar school om uitleg te geven over verschillende richtingen en ik had naar een paar geluisterd en had hierdoor wat meer ideeën voor wat ik wow volgen. Door heen het jaar had ik de keuzes naar 2 keuzes uitgewerkt door persoonlijke interesse het was tussen: interieur architect en programmeren. Eerst keek ik voor info voor interieur architect op de website van UHasselt, maar ik vond hier niets meer op terug dus toen ik voor programmeren keek, vond ik wel info op de PXL-website en koos ik uiteindelijk om programmeren te gaan volgen.

Past dit beroep bij mij?

ik heb altijd interesse gehad in de achterliggende code van dingen en hoe deze in elkaar zit en ik heb liever een job waar ik veel kan zitten (door medische redenen) ook vind ik het fijn om code te schrijven en vooral als het werkt ook heb ik geen probleem met door code te gaan voor uren om fouten op te sporen of een efficiëntere manier te vinden. Ook ben ik redelijk goed in dingen waarvan een normaal mens niets van weet uit te leggen op een manier dat ze het wel snappen. En ik ben door mijn werkgeschiedenis goed met elke soort klant om te gaan van goed tot slecht.
