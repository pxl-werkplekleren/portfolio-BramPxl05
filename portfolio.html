Waarom heb ik voor deze richting gekozen?

Door heen het jaar had ik de keuzes naar 2 keuzes uitgewerkt door persoonlijke interesse het was tussen: interieur architect en programmeren. Past dit beroep bij mij?

ik heb altijd interesse gehad in de achterliggende code van dingen en hoe deze in elkaar zit en ik heb liever een job waar ik veel kan zitten (door medische redenen) ook vind ik het fijn om code te schrijven en vooral als het werkt ook heb ik geen probleem met door code te gaan voor uren om fouten op te sporen of een efficiëntere manier te vinden. Ook ben ik redelijk goed in dingen waarvan een normaal mens niets van weet uit te leggen op een manier dat ze het wel snappen. En ik ben door mijn werkgeschiedenis goed met elke soort klant om te gaan van goed tot slecht.
